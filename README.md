# Development board

# Peripheral

With the development board,stm32 chip can provide follwing function pins:

1. SWD
2. ADC
3. DAC
4. FMC
5. LCD
6. SPI
7. I2C
8. SDMMC
9. USB
10. USB-VPC
11. USB-TypeC
